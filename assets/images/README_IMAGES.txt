==========================================
  PETUNJUK UPLOAD GAMBAR
==========================================

Silakan upload gambar-gambar berikut ke folder ini:

1. logo-sekolah.png
   - Logo sekolah (ukuran 200x200px, format PNG dengan background transparan)
   - Letakkan di: assets/images/logo-sekolah.png

2. hero.jpg
   - Gambar untuk hero/banner utama (ukuran 1920x1080px)
   - Letakkan di: assets/images/hero.jpg

3. Foto Kepala Sekolah dan Pimpinan
   - Upload melalui admin panel di menu "Pimpinan"
   - Akan otomatis tersimpan di: assets/images/pimpinan/

4. Foto Guru
   - Upload melalui admin panel di menu "Tenaga Pendidik"
   - Akan otomatis tersimpan di: assets/images/guru/

5. Gambar Berita
   - Upload melalui admin panel di menu "Berita"
   - Akan otomatis tersimpan di: assets/images/berita/

6. Foto Prestasi
   - Upload melalui admin panel di menu "Prestasi Siswa"
   - Akan otomatis tersimpan di: assets/images/prestasi/

7. Galeri
   - Upload melalui admin panel di menu "Galeri"
   - Akan otomatis tersimpan di: assets/images/galeri/

==========================================
CATATAN PENTING:
- Semua gambar akan otomatis di-resize saat upload
- Ukuran maksimal per file: 5MB
- Format yang diperbolehkan: JPG, PNG, GIF
- Pastikan folder memiliki permission 777 (writable)
